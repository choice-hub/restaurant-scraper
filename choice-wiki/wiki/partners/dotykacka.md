## About Dotykačka
One of the most widely used cloud POS systems in CZ and SK. Part of the **Dotypos Group**. Used by thousands of gastro venues — cafés, restaurants, bistros, hotels.

## Choice Integration
Direct API integration between Dotykačka and Choice QR — stable, fully automated.

**What it enables:**
- Orders from Choice (QR, takeaway, delivery) flow directly into Dotykačka POS — no manual entry
- Items marked in Dotykačka appear on the QR code — guests can pay via QR
- Table and account sync via POS ID
- Two-way order and payment data

**Important:** Integration is only available on **Standard and Profi** plans — not on the Start (entry) plan.

**Technically:** API keys + item/table ID mapping. No client-side dev work needed.

## Dotykačka System Features
- Cloud POS with multi-device support
- Mobile waiter: orders and payments at table
- **Dotypay payment terminal:** integrated with POS, supports Apple Pay + Google Pay
- Smart tipping: tip management and payout overview
- Customer portal: online access to sales data, settings, stats
- Advanced BI reporting (custom dashboards, individual analytics)
- Inventory management (higher plans)
