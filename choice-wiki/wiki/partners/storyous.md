## About Storyous
Czech cloud POS system for restaurants, cafés, bars. Used by 5,000+ venues in CZ and SK — one of the most widely deployed cloud POS systems in the region.

## Choice Integration
The Storyous ↔ Choice integration is one of the most complete and reliable on the market.

**What it enables:**
- Orders from Choice (QR, takeaway, delivery) flow directly into Storyous POS — no manual entry
- Items marked in Storyous POS appear on the QR code — guests can pay via QR
- Table and product sync via POS ID
- Two-way data: order status, bill closing

**Technically:** API key + item/table ID mapping. Stable, fast, no dev work required on the client side.

## Storyous System Features
- Cloud — accessible anywhere, offline mode available
- Multi-register: all locations managed from one admin
- Real-time inventory — automatic deductions, stocktaking
- Mobile waiter — all-in-one device for orders, bill splitting, payments at table
- Auto-restocking: direct connection to wholesalers
- 24/7 support
