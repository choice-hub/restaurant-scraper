# Product: Restaurant Website

## Overview
Choice builds and hosts a fully branded restaurant website with an integrated online ordering system. Restaurants can be live in **48 hours** with no developer or designer needed.

## Key Features

- **Mobile-first design** — 90% of users browse via mobile; all Choice sites are fully responsive
- **Integrated ordering** — Delivery and takeaway orders built in; no third-party plug-ins needed
- **Custom domain** — Use your own domain (e.g. myrestaurant.com)
- **SEO optimized** — Built-in SEO tools, meta tags, Google integration
- **99-language auto-translation** — Serve international guests automatically
- **Promotional banners** — Highlight deals, happy hours, seasonal offers
- **POS integration** — Orders flow directly to Poster, Syrve, R-Keeper, etc.
- **Payment methods** — Bank cards, Apple Pay, Google Pay
- **Multi-location support** — One admin panel for all branches
- **Review system** — Customers can leave feedback post-order
- **Analytics** — Order volume, average check, popular dishes

## Customization
- Logo, brand colors, custom fonts
- Page block management (reorder, show/hide sections)
- Guest info blocks (Wi-Fi password, parking, story)
- Integration with Google Analytics and Facebook Pixel
