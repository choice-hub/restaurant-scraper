# Product: Marketing Tools

## Overview
An integrated set of marketing tools to help restaurants attract guests, drive repeat visits, and measure what's working — all from the same admin panel.

## Key Features

**Push Notifications & Email**
- Send real-time promotional notifications to customers with the app or account
- Email campaigns for menu updates, special offers, events
- Automated retention notifications (e.g. "We miss you" after 30 days of inactivity)

**SEO Optimization**
- Built-in SEO tools for the restaurant website
- Local SEO optimization (Google Maps visibility)
- Keyword suggestions and meta tag management
- Google Search Console integration

**Analytics Dashboard**
- Website traffic and visitor behavior
- Order conversion rates per channel
- Campaign performance metrics
- Best-seller dish analysis
- Channel comparison (direct vs. marketplace)

**Third-party Integrations**
- Google Analytics
- Facebook Pixel
- Google Tag Manager
- Google Search Console

**AI Marketing Tools** *(Pro plan)*
- AI Text edit and translations
- Marketing content generator — create posts, campaigns, and promotions
- Review reply assistant — AI-generated responses to customer feedback
- Analytics AI assistant — ask questions about restaurant performance

## Business Impact
| Metric | Improvement |
|--------|------------|
| Online reservation increase | +40% |
| Revenue from promotions | Up to +25% |
| Customer retention improvement | +35% |
| Positive reviews increase | +50% |

## Plans
Basic marketing tools (analytics, SEO, email) available on all plans.
Advanced tools (SMS, loyalty campaigns) on Smart and Pro.
AI tools on **Pro plan only**.
